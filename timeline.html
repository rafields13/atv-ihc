Timeline do Docker

2008

2008 - Cgroups e Namespaces

O kernel do Linux introduz os cgroups e namespaces, possibilitando o isolamento de processos e recursos, base para a criação de containers.

2013

2013 - Nascimento do Docker

A Docker Inc. lança o Docker como projeto open source, oferecendo uma forma prática de empacotar e distribuir aplicações em containers.

2014

2014 - Docker Hub

É lançado o Docker Hub, um repositório público para imagens de containers, facilitando o compartilhamento de aplicações.

2015

2015 - Docker Compose & Swarm

São lançados o Docker Compose, para definir aplicações multicontainer, e o Docker Swarm, para orquestração de containers em cluster.

2016

2016 - Kubernetes

Kubernetes se torna o orquestrador padrão da indústria, com suporte a Docker, promovendo escalabilidade e resiliência em ambientes de produção.

2020

2020 - Docker no DevOps

Adoção massiva do Docker em pipelines CI/CD e ambientes de desenvolvimento integrados com GitHub Actions, GitLab CI e outros.

2021

2021 - Docker Desktop para M1

A Docker lança suporte oficial para chips Apple Silicon (M1), ampliando compatibilidade e desempenho para desenvolvedores.

2023

2023 - Docker + Kubernetes

Docker e Kubernetes seguem como ferramentas essenciais em ambientes corporativos, cloud híbrida e microserviços.
