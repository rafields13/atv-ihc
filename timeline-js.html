Timeline JavaScript

1995

1995 - Criação do JavaScript

Brendan Eich cria a linguagem inicialmente chamada Mocha, que evolui para JavaScript, trazendo interatividade ao Netscape Navigator.

1996

1996 - Microsoft lança o JScript

A Microsoft cria o JScript para o Internet Explorer 3, gerando as primeiras incompatibilidades entre navegadores.

1997

1997 - ECMAScript 1

A padronização do JavaScript pela ECMA International como ECMAScript 1 abre caminho para a compatibilidade entre navegadores.

1999

1999 - ECMAScript 3

A versão ECMAScript 3 traz melhorias significativas como expressões regulares e manipulação avançada de strings.

2005

2005 - AJAX

A introdução do AJAX revoluciona a web ao possibilitar atualizações dinâmicas sem recarregar a página.

2006

2006 - jQuery

O lançamento do jQuery simplifica a manipulação do DOM, tornando o desenvolvimento web mais acessível.

2009

2009 - Node.js

Ryan Dahl lança o Node.js, permitindo que o JavaScript seja utilizado também no ambiente de servidor.

2012

2012 - AngularJS

O AngularJS surgiu como um framework de JavaScript de código aberto desenvolvido por Misko Hevery e Adam Abrons, enquanto trabalhavam no Google. Seu objetivo era facilitar a criação de aplicativos web dinâmicos e robustos, usando uma abordagem baseada em MVC (Model-View-Controller)

2015

2015 - ECMAScript 6

O ES6 moderniza o JavaScript com novas funcionalidades, como let, const, arrow functions, classes e promises.

2016

2016 - React e Vue.js

Frameworks como React e Vue.js ganham força, promovendo um desenvolvimento front-end mais modular e eficiente.

2019

2019 - ECMAScript Anual

A partir desta data, o JavaScript passa a receber atualizações anuais, garantindo a evolução contínua da linguagem.

2023

2023 - Linguagem Mais Popular

Pelo 11º ano consecutivo, o JavaScript lidera as pesquisas de linguagens de programação, sendo essencial em diversas áreas do desenvolvimento.
